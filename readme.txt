== Description ==
  
**Summary:**

Advanced Product Filter For WooCommerce is a product search plugin for WooCommerce that enables users to filter products on your website by price, categories, characteristics, tags, and custom lists. It is a necessary plugin for your online store powered by WooCommerce! maximal adaptability :)

We offer new styles for filters: Checkbox, Select, Slider, Color, Radio, Search Box.

Demo: [demofilter.vrinsoft.in](https://demofilter.vrinsoft.in/)

**Features:**

&#9989;&nbsp;**1.** It has an amazing representation option for the products.
&#9989;&nbsp;**2.** There are product shortcut codes for easy integration.
&#9989;&nbsp;**3.** Products searching will list out products without page loading.
&#9989;&nbsp;**4.** Gives powerful and dynamic search to the users.
&#9989;&nbsp;**5.** The products can be displayed by the sliders.
&#9989;&nbsp;**6.** Helps in Quick search and displays both quality and quantity products.
&#9989;&nbsp;**7.** The filter gives infinite scrolling for the products and services.
&#9989;&nbsp;**8.** Easy integration of layout and template in the filter.
&#9989;&nbsp;**9.** Developer friendly plugin

**Additional Features:**

&#9989;&nbsp;**1.** Table-format product: It displays the products using the WooCommerce plugin in tables. Your customers can concentrate entirely on what is being presented and not on anything extraneous or unnecessary.

&#9989;&nbsp;**2.** WordPress content in articles and their custom types. Thanks to its extensive filter components and built-in metadata, the plugin provides a very high degree of versatility.

&#9989;&nbsp;**3.** If you are a WP, or PHP developer and wish to make an interesting search form interface. You can develop any HTML items for the search form.

**Process Points:**

&#9989;&nbsp;**1.** Download and install using the WordPress admin interface or the plugin directory.
